# KNVOX V1.4.0 - Customer Admin API

## Objectif

Cette version ajoute la gestion opérationnelle des clients call center.

## Fonctions

- Création client
- Liste clients
- Détail client
- Crédit solde prépayé
- Modification CPS
- Modification appels simultanés
- Modification plafond journalier
- Modification tarif maximum autorisé
- Suspension / réactivation
- Fraud lock / unlock
- Consultation CDR
- Consultation décisions routing
- Consultation événements fraude

## Sécurité

L'API reste interne sur 127.0.0.1.

Le PSTN reste désactivé.

Aucun trunk fournisseur réel ne doit être connecté dans cette version.
