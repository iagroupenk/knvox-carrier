# KNVOX V1.4.1 - Rate & Prefix Admin API

## Objectif

Gérer les tarifs, préfixes bloqués et routes fournisseurs sandbox.

## Fonctions

- Lister les tarifs de vente
- Ajouter/modifier un tarif de vente
- Désactiver un tarif
- Lister les préfixes bloqués
- Ajouter un préfixe bloqué
- Supprimer un préfixe bloqué
- Ajouter/modifier une route fournisseur sandbox
- Journaliser les actions tarifaires

## Sécurité

Le PSTN reste désactivé.

Les fournisseurs créés restent en mode sandbox.

Aucun trunk réel ne doit être connecté dans cette version.
